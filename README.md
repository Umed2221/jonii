# Онлайн школа рисования

Полностью переписанная статическая версия сайта онлайн школы рисования.

## Что внутри

- `index.html` - разметка сайта.
- `css/styles.css` - адаптивные стили.
- `js/app.js` - вся интерактивная логика.
- `data/lessons.js` - уроки, категории, идеи и статьи.
- `assets/images/` - локальные изображения.
- `server.js` - простой локальный сервер без установки зависимостей.
- `vercel.json` - настройки для Vercel.
- `_redirects` - настройки для Netlify.

## Локальный запуск

```bash
npm run dev
```

Откройте:

```text
http://127.0.0.1:5173/
```

Устанавливать пакеты не нужно.

## Деплой

### Vercel

1. Загрузите папку проекта в GitHub.
2. Создайте проект в Vercel.
3. Framework Preset: `Other`.
4. Build Command: `npm run build` или оставьте пустым.
5. Output Directory: оставьте пустым.

### Netlify

1. Загрузите папку проекта.
2. Build command: пусто или `npm run build`.
3. Publish directory: корень проекта.

### Обычный хостинг

Загрузите все файлы из этой папки в корень сайта. Главный файл должен лежать как `index.html`.

## Админ-доступ

Тестовый вход:

```text
admin@drawing.com
admin123
```

Данные и прогресс сохраняются локально в браузере через `localStorage`.
